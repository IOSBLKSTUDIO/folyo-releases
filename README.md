# Folyo Releases

Distribution publique des binaires Folyo — logiciel de comptabilité macOS.

> Le **code source** de Folyo est dans un dépôt **privé** (propriété d'IOS BLK Studio).
> Ce dépôt-ci ne contient que les **DMG signés et notarisés Apple** prêts à l'installation.

## Téléchargement

- 🌐 Site officiel : **https://folyoapp.fr**
- 📦 Dernière version stable : voir l'onglet [Releases](../../releases/latest)
- 🔄 URL stable du DMG : `https://github.com/IOSBLKSTUDIO/folyo-releases/releases/latest/download/Folyo-arm64.dmg`

## Configuration requise

- macOS 12 Monterey ou supérieur
- Mac Apple Silicon (M1, M2, M3, M4)
- ~500 Mo d'espace disque libre

## Sécurité

Tous les DMG sont :
- Signés avec un certificat Apple Developer ID
- Notarisés par Apple à chaque release
- Aucune alerte Gatekeeper au lancement

## Auto-update

L'application gère automatiquement les mises à jour via electron-updater. Aucune action manuelle requise après l'installation initiale.
